# Mochi Farm 

## Overview

Official frontend repo of [Mochi.Farm](https://app.Mochi.Farm).Yield farming optimizer.

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 288 // Network ID of BOBA
```

## To Run

```
yarn
yarn start
```

The app will be running at http://localhost:3000/
